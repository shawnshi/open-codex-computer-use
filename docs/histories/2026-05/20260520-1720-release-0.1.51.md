# 发布 0.1.51

## 背景

权限引导浮层修复已经合入 `main`，需要发布新的插件和 CLI 版本，让全局安装用户拿到 `PermissionDiagnostics.current()` 的权限完成态修复。

## 主要改动

- 将插件 manifest、Swift 版本源、smoke fixture、Linux / Windows CLI、computer-use helper CLI 与 README 示例版本同步到 `0.1.51`。
- 在功能发布记录中新增 `0.1.51` 条目，说明 macOS 权限引导浮层完成后不再持续显示。
- 准备基于合并后的 `main` 创建 `v0.1.51` annotated tag，触发 GitHub release workflow。
